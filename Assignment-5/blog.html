Q: What is DOM and what is the purpose of DOM?

A: The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

Q: How will you select HTML elements using DOM? Name the DOM methods.

A: There are 5 ways in which you can select elements in a DOM using selectors.

  1. getElementById()
  2. getElementsByClassName()
  3. getElementsByTagName()
  4. querySelector()
  5. querySelectorAll()

There are six different methods in which users can access or manipulate HTML elements using JavaScript:

  • HTML DOM getElementById() Method.
  • HTML DOM getElementsByClassName() Method.
  • HTML DOM getElementsByName() Method.
  • HTML DOM getElementsByTagName() Method.
  • HTML DOM querySelector() Method.

Q: What is event bubble?

A: Event Bubbling is a concept in the DOM (Document Object Model). It happens when an element receives an event, and that event bubbles up (or you can say is transmitted or propagated) to its parent and ancestor elements in the DOM tree until it gets to the root element.

Q: What is a callback function and why will you use it?

A: A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action.
